Quick checklist for “giving access” to a page

1. Is the page a client component?

  - Add "use client" at the top.

  - Call const { user, session, loading } = useSupabaseAuth();

  - Use user.email, user.id, etc.

2. Is the page a server component?

  - Use your createClient() with cookies.

  - Call supabase.auth.getUser() on the server.

  - Query any user-specific data (like profiles) using user.id.

3. Protecting a page

  - On client: if !user && !loading → show “sign in” or router.push("/Login").

  - On server: if !user → redirect("/Login").
