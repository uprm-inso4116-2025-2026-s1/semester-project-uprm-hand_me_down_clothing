=== *2.2.3 Domain Requirements*

[requirements]

[[REQ-DR1]]
*DR1:* The system must classify every listing under exactly one primary category.  
*Supported by:* <<FEATURE-1.1>>, <<FEATURE-1.2>>, <<FEATURE-1.3>>, <<FEATURE-2.1>>, <<FEATURE-2.2>>, <<FEATURE-3.4>>, <<FEATURE-4.1>>, <<FEATURE-4.2>>, <<FEATURE-4.3>>, <<FEATURE-6.2>>

[[REQ-DR2]]
*DR2:* Categories may have hierarchical subcategories.  
*Supported by:* <<FEATURE-1.1>>

[[REQ-DR3]]
*DR3:* The system must distinguish between Resale and Free/Donation listings.  
*Supported by:* <<FEATURE-1.2>>, <<FEATURE-3.1>>, <<FEATURE-3.2>>, <<FEATURE-3.3>>, <<FEATURE-4.4>>, <<FEATURE-5.1>>, <<FEATURE-5.2>>, <<FEATURE-6.1>>, <<FEATURE-6.2>>

[[REQ-DR4]]
*DR4:* The system shall enforce item-to-category compatibility.  
*Supported by:* <<FEATURE-1.1>>, <<FEATURE-4.1>>

[[REQ-DR5]]
*DR5:* The system must support category evolution.  
*Supported by:* <<FEATURE-4.3>>
