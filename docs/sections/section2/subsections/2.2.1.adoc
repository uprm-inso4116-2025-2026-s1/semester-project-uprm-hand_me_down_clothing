=== *2.2.1 Epics, Features, and User Stories*

Epics describe high-level goals that span multiple sprints and require several coordinated features.  
Features refine epics into concrete functionality.  
User stories express these features from the perspective of stakeholders within the domain.

All EPIC → FEATURE and FEATURE → REQUIREMENT traceability links use AsciiDoc anchors for clarity.

---

== *Epics*

=== *Buyer Epics*

[[EPIC-1]]
. **Listing Discovery**  
As a buyer, I want to browse, filter, and search listings so that I can efficiently find items that match my preferences and financial constraints.  
*Implemented by:* <<FEATURE-1.1>>, <<FEATURE-1.2>>, <<FEATURE-1.3>>

[[EPIC-2]]
. **Saved Listings**  
As a buyer, I want to save or bookmark listings so I can revisit them later when deciding whether to contact the seller.  
*Implemented by:* <<FEATURE-2.1>>, <<FEATURE-2.2>>

[[EPIC-3]]
. **Trust and Transparency**  
As a buyer, I want access to seller profiles and post-exchange reviews so I can make informed decisions before initiating contact.  
*Implemented by:* <<FEATURE-3.1>>, <<FEATURE-3.2>>, <<FEATURE-3.3>>, <<FEATURE-3.4>>

---

=== *Seller Epics*

[[EPIC-4]]
. **Listing Management**  
As a seller, I want to create, update, and close listings so I can manage items I intend to resell, donate, or withdraw.  
*Implemented by:* <<FEATURE-4.1>>, <<FEATURE-4.2>>, <<FEATURE-4.3>>, <<FEATURE-4.4>>

[[EPIC-5]]
. **Interest Notifications**  
As a seller, I want to receive notifications when buyers express interest so I can respond promptly and arrange offline exchanges.  
*Implemented by:* <<FEATURE-5.1>>, <<FEATURE-5.2>>

[[EPIC-6]]
. **Seller Profile and Trust Signals**  
As a seller, I want to maintain a profile with minimal, privacy-respecting details so buyers can build confidence before contacting me.  
*Implemented by:* <<FEATURE-6.1>>, <<FEATURE-6.2>>

== *User Stories*

User stories refine features by describing concrete goals from the perspective of domain actors.

=== *Buyer User Stories*

- As a buyer, I want to browse by category so I can quickly find the kind of item I need.  
- As a buyer, I want to filter by size, condition, or price/free marker so my results match my constraints.  
- As a buyer, I want to search using keywords so I can locate specific items not covered by filters.  
- As a buyer, I want to save interesting listings so I can revisit them later.  
- As a buyer, I want to view essential seller information (rating, location, account age) while still respecting seller privacy.  
- As a buyer, I want to leave a review after an offline exchange so others can benefit from my experience.

=== *Seller User Stories*

- As a seller, I want to create listings with clear details and images so buyers can evaluate items quickly.  
- As a seller, I want to edit my listings so I can keep them accurate.  
- As a seller, I want to close listings by marking them as *sold*, *donated*, or *retracted* so my dashboard remains up-to-date.  
- As a seller, I want to receive notifications when someone expresses interest so I can coordinate the exchange.  
- As a seller, I want to present minimal but trustworthy profile information to attract buyers without oversharing.
